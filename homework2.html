Introduction

This page explains what a dataset and a distribution are, shows a very small example dataset created for the assignment, and computes three univariate distributions. A short bivariate view is added at the end as an extra step.

What a Dataset Is

A dataset is a structured collection of observations. Each row represents one observation and each column represents a variable that describes that observation. Datasets make information analyzable and reproducible, because values and definitions are explicit.

What a Distribution Is

A distribution describes how the values of a variable are spread across its possible outcomes. For categorical variables it tells how often each category appears. For numerical variables it summarizes the frequency of values and helps describe location and spread.

Conclusion

The dataset provides a clear structure for analysis. Distributions translate that structure into evidence about typical behavior and unusual events. Even a small sample can guide attention, while a larger one would allow stronger conclusions and automation through standard statistical tools or simple classifiers.
