Homework 8 — Analogies Between LLN Bernoulli Simulation and Random Walk

Konstantinos Tziakouris • Matricola 2229757 • Statistics • A.Y. 2025 2026

1) Connection With Previous Homeworks

In a previous homework on the Law of Large Numbers, we simulated a sequence of independent Bernoulli trials. Each trial produced result 1 for success and 0 for failure, and we studied the empirical mean f(n) = (number of successes) / n, showing that it converges to the true probability p.

In Homework 7 the same Bernoulli structure appears in a different form. The server status each week is represented by a random variable Xi ∈ {+1, -1}, with P(Xi = +1) = 1 − π and P(Xi = −1) = π, where π = 1 − (1 − p)m is the weekly breach probability. The cumulative score of the server is the random walk Sn = ∑ Xi.

Both simulations are based on independent Bernoulli trials. The LLN homework focuses on averages, the random walk homework on cumulative sums.

2) Common Bernoulli Structure and Mapping

Let Bn be the number of breaches in n weeks. Then Bn ∼ Binomial(n, π). The score process of Homework 7 can be written as Sn = (#secure weeks) − (#breached weeks) = (n − Bn) − Bn = n − 2Bn.

If we define the empirical breach frequency f(n) = Bn / n, then Sn / n = 1 − 2 f(n). The LLN tells us that f(n) converges to π, so Sn / n converges to 1 − 2π. This links the Law of Large Numbers simulation directly to the behavior of the random walk.

4) Binomial Coefficients and Pascal Triangle

In both simulations the key counting object is the number of successes in n independent trials. For the server model this is the number of breached weeks Bn, which follows Binomial(n, π) with probability mass function

P(Bn = k) = C(n, k) πk (1 − π)n − k, k = 0, 1, ..., n.

The coefficients C(n, k) are the entries of Pascal triangle. Each level n of the triangle lists all the binomial coefficients for that number of trials and counts how many distinct paths have exactly k breaches.

In Homework 7 the histogram of final scores is just the binomial distribution of Bn, written on the score scale Sn = n − 2k.

5) Binomial Expansion and Probability Space

Pascal triangle also encodes the coefficients in the binomial expansion

(a + b)n = ∑ C(n, k) an − k bk.

In the probabilistic setting of the server model we take a = 1 − π (secure week), b = π (breached week). The identity

(1 − π + π)n = ∑ C(n, k) (1 − π)n − k πk = 1

shows that the binomial coefficients and powers of π and 1 − π describe the entire probability space of outcomes. Each term corresponds to one bar in the histogram of the simulation, and the code in Homework 7 is effectively reconstructing this expansion numerically.

6) Fibonacci and Constrained Paths

The combinatorial viewpoint can be pushed further by adding constraints on the paths. If we forbid two breaches in a row or do not allow the walk to cross below zero, the number of admissible trajectories is no longer given only by binomial coefficients.

  • If we forbid consecutive breaches, the number of valid binary strings of length n without two ones in a row follows a Fibonacci type recursion.
  • If we require that the walk never goes below zero, then the number of valid paths of length 2n is given by Catalan numbers Cn = (1 / (n + 1)) C(2n, n).

Even very simple decisions of type plus one or minus one can hide rich combinatorial structures.

7) LLN, Variance and Central Limit Theorem

In the LLN homework the main quantity was the empirical mean of Bernoulli trials. Its variance is Var(Bn / n) = p (1 − p) / n, which decreases with n and explains the stabilization around p.

In the random walk homework the main quantity is the cumulative score Sn. Using the relation with Bn we have E[Sn] = n (1 − 2π) and Var(Sn) = 4 n π (1 − π). After scaling by n or by √n, both models satisfy the Central Limit Theorem and lead to Gaussian limits.

8) Conceptual Takeaway

The simulation of the Law of Large Numbers and the security random walk describe the same Bernoulli world from two different angles. One looks at averages and proportions, the other at cumulative signed scores and trajectories. In both cases the same binomial coefficients, Pascal triangle and binomial expansion appear in the background, and the same limit theorems explain why the random behavior becomes predictable when the number of trials is large.
